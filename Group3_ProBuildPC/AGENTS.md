# AGENTS.md

## Project Overview

This project is a final-term computer shop website.

Technology stack:
- Java JSP/Servlet
- HTML
- CSS
- SQL Server or MySQL
- NetBeans

Architecture:
- MVC pattern
- DAO pattern

## Main Features

Customer:
- Register
- Login
- Logout
- View products
- Search products
- Add to cart
- Checkout
- View order history

Admin:
- Manage products
- Manage categories
- Manage brands
- Manage users
- Manage orders

## Coding Rules

- Keep code simple and understandable.
- Do not over-engineer.
- Use MVC architecture.
- Use DAO for database operations.
- JSP should only display data.
- Do not put SQL inside JSP.
- Use meaningful variable names.
- Follow existing project structure.

## Folder Structure

- controller/
- dao/
- model/
- service/
- utils/

## Before Writing Code

Always:
1. Read requirements.
2. Explain implementation plan.
3. Keep changes minimal.
4. Summarize modified files.

## Do Not

- Do not use Spring Boot.
- Do not delete existing files without explanation.
- Do not change database structure unexpectedly.
- Do not hardcode passwords.
